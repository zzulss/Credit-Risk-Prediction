# Credit Risk Prediction

## Objective

Develop a machine learning model to predict whether a customer is likely to default on a loan.

---

## Problem Statement

Financial institutions need to identify high-risk borrowers before approving loans. This project compares several machine learning models to improve credit risk prediction.

---

## Dataset

German Credit Dataset

Source:
https://archive.ics.uci.edu/ml/datasets/statlog+(german+credit+data)

---

## Project Workflow

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Model Explainability (SHAP)
